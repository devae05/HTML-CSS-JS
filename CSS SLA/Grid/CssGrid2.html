CSS Grid Lines The lines between columns are called column lines.The lines between rows are called row lines. 

We can specify where to start and end a grid item by using the following properties:


grid-column-start   grid-column-end
grid-row-start      grid-row-end
grid-column         grid-row

The grid-column-start and grid-column-end Properties

The grid-column-start property specifies where to start a grid item. The grid-column-end property specifies where to end a grid item.
