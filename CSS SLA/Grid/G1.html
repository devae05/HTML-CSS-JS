CSS Grid Layout

What is CSS Grid?

CSS Grid is a 2-dimensional layout system (rows & columns) used to design complex web layouts easily.

Unlike Flexbox (1-D), Grid works in 2-D (rows + columns).

Why Use CSS Grid?

Create complex layouts easily
Align items both horizontally & vertically
Responsive design without media queries
Clean & readable CSS
